# Простой сайт (demo)

В этом архиве:
- `index.html` — главная страница
- `about.html` — страница "Обо мне"
- `contact.html` — страница контактов (форма демо)
- `css/styles.css` — стили
- `js/script.js` — простой скрипт
- `images/` — папка с SVG-изображениями и favicon

Как запустить:
1. Распакуйте архив.
2. Откройте `index.html` в браузере.
3. Для работы формы или отправки — нужен сервер (пример: `python -m http.server` в каталоге сайта).
